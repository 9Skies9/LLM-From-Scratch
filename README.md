# LLM From Scratch - Personal HW + Notes

Notes + homework follow-along for studying LLMs from scratch.

This repository contains my personal chapter-by-chapter follow-along work based on the original materials.

## Original Course Attribution
The original learning material and chapter structure come from Sebastian Raschka's official resources:
- Source repository: [rasbt/LLMs-from-scratch](https://github.com/rasbt/LLMs-from-scratch)
- Main book: [Build a Large Language Model (From Scratch)](https://amzn.to/4fqvn0D)

I do not claim the original content as my own. This repository is my personal learning implementation, notes, and homework-style practice while following that material.

## Repository Layout
Current top-level structure:

- `1 Notes/` - Personal chapter notes (`.md`) and note attachments
- `2 Code/` - Notebook/code implementations by chapter
- `LLMs-from-scratch/` - Local upstream reference clone (excluded from upload)

## Environment Setup
1. Install Python 3.
2. Create and activate a virtual environment.
3. Install notebook and ML dependencies as needed (for example: Jupyter, PyTorch, NumPy, pandas, matplotlib, tiktoken).

## Running Notes
- Most chapter work is in Jupyter notebooks under `2 Code/`.
- Notes are organized by chapter under `1 Notes/`.
- Large local model checkpoints and local reference copies are intentionally excluded from git for cleaner publishing.
